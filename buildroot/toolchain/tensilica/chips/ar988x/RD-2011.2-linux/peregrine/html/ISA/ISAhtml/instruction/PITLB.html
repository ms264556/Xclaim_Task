PITLB — Probe Instruction TLB

Instruction Word

x24 : x24_Inst_0
2 3 1 2 1 1 8 7 4 3 0
12 4 4 4
0 1 0 1 0 0 0 0 0 1 0 1 s t 0 0 0 0

Assembler Syntax

PITLB at, as

C Syntax

extern unsigned XT_PITLB(unsigned ars);

Description

(please consult the Xtensa ® Instruction Set Architecture Reference Manual for any cross references and additional information)

PITLB searches the Instruction TLB for an entry that translates the virtual address in address register as and writes the way and index of that entry to address register at. If no entry matches, zero is written the Hit bit of at. The value written to at is implementation specific, but in all implementations a value with the Hit bit set is suitable as an input to the IITLB or WITLB instructions. See Section 4.6.3.2 in the Xtensa ® Instruction Set Architecture Reference Manual for more information, Section 4.6.4.2 in the Xtensa ® Instruction Set Architecture Reference Manual for more information, and Section 4.6.5.7 in the Xtensa ® Instruction Set Architecture Reference Manual for more information for more information on the result register formats for specific Memory Protection and Translation options.

PITLB is a privileged instruction.

Operation

if CRING ≠ 0 then
	Exception (PrivilegedInstructionCause)
else
	(match, vpn, ei, wi) ← ProbeInstTLB(AR[s])
	if match > 1 then
		EXCVADDR ← AR[s]
		Exception (InstructionFetchTLBMultiHit)
	elseif match = 0 then
		AR[t] ← PackInstTLBEntrySpec(1"b0, vpn, ei, wi)
	else
		AR[t] ← PackInstTLBEntrySpec(1"b1, vpn, ei, wi)
	endif
endif

Exceptions

  • EveryInstR Group (see the Xtensa ® Instruction Set Architecture Reference Manual for more information)
  • GenExcep(PrivilegedCause) if Exception Option
